# Book_notes

Simple Book Notes app (Node.js + Express + EJS) that fetches book metadata from Open Library and stores notes/ratings.

Features

- Search and add books via Open Library
- Display book cover, author, rating
- Remove books

Run locally

1. Install dependencies:

```bash
npm install
```

2. Start server (development):

```bash
npm run dev
```

3. Open http://localhost:3000

Notes

- Create a `.env` or configure PostgreSQL in `index.js` if you want persistent storage.
- Add your `.env` to `.gitignore` (already included below).
